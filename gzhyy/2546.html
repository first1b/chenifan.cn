从0到1，为新公众号寻找市场突围点和运营方向

运营人—tem365.com 公众号运营 发布时间：2020-12-29 0

背景目的：

假设我是「十点读书」的新媒体运营，现在计划从0到1做一个旅行类的公众号，并负责这个号的运营工作。

我现在需要先了解竞品的情况，根据几个比较大的旅行类公众号的推文数据以及推文内容绘制出积木模型和better-worse模型，来判断是否还有市场突围的可能性。然后寻找该号的突围点并确定运营方向。

工具方法：

  1. 产品生命周期理论
  2. 积木模型
  3. better-worse模型

具体步骤：

  1. 调研十点文化基本信息和「十点读书」公众号的情况；
  2. 分析目前旅游行业现状及十点读书在旅游方向的业务尝试；
  3. 对竞品公众号的用户、内容进行调研，发现竞品的运营特点；
  4. 通过竞品分析，绘制积木模型，寻找十点读书旅行号的突围点；
  5. 利用better-worse模型，确定十点读书旅行号的运营方向和运营建议。

一、十点读书分析

1. 基本信息

1. 公司信息

十点读书是厦门十点文化传播有限公司旗下的基于微信公众号的读书分享自媒体平台，业务范围覆盖文学、电影、 时尚、教育、出版、实体书店等多个领域。

旗下产品包括十点读书、十点课堂、十点读书会、十点电影、十点书店、小十点、她读、十点、十点好物、人物志等公众号，通过图文、音频、视频、图书、社群等多种形态将原创内容产品传递给用户。

2. 发展历程

十点读书起步于微博，生长于微信，顺应媒体渠道的变化和迁移趋势，不断迭代文化生活品牌。

*图片来源：《“十点读书”用户从0到4000万：机械工科男的内容创业心得》——中传何教授

3. 商业变现

十点的产品矩阵以“十点读书”为中心，辐射到多个领域，如付费课程、电商、视频等，用丰富的产品去服务全网4000万用户，不断寻找新的机会、渠道和平台。

目前“十点读书”的商业变现方式是在线课程（十点课堂50%）+ 电商（十点小店）+ 品牌广告（20%）+ 书籍出版。

2018年12月，十点读书全国首家十点书店在厦门正式开业，可见十点读书在进一步探索变现的方式。

2. 公众号属性

1. 简介及粉丝量

从西瓜助手「十点读书」公众号的简介可以看出，目前活跃粉丝数预计超过2000万，拥有庞大的粉丝量。

2. 生命周期

「十点读书」公众号活跃粉丝超过100W，根据产品生命周期理论，可知目前处于成熟期，现阶段账号基本定型，增长放缓，核心关注点是品牌和商业变现，需要通过建立矩阵来细分人群。

3. 用户画像

利用西瓜助手对「十点读书」公众号进行用户画像诊断，发现用户年龄集中在18-29岁，其中女性占70%，即主要是90后。

在用户族群分布上，各族群占比差异不大，反映出【十点读书】用户族群的多样性、平衡性。

4. 内容分析

在传送门上爬取「十点读书」近一周（2019.1.1-2019.1.7）的56个文章标题进行分析。

1. 内容主题

热点新闻、个人成长、婚姻情感、人物故事、生活方式以及家庭教育。

2. 标题特点

会紧扣时事热点吸引读者关注，多以疑问式标题引发读者思考，以及引导式标题指导读者行动。

3. 文章风格

抒情、叙事、议论，多为治愈系暖文，用文字给读者带来温暖和力量。

二、旅游业务分析

1. 旅游行业动向

根据《360：2018出行旅游行业洞察报告》——119IT 可知 ：

  1. 我国在线旅游持续渗透，市场规模稳定增长；
  2. 移动端旅游搜索更聚焦，目的地、国家词占比最高；
  3. 旅游相关网站、游记攻略、航空是旅游行业整体最关注的内容；
  4. 境外游越来越受关注，境内游仍是主力；
  5. 周边游、亲子游、短线游受欢迎，自助游热度持续升高。

2. 旅游人群画像

根据 《360：2018出行旅游行业洞察报告》——199IT 可知 ：

  1. 男性更关注机酒和路线，女性更关注旅游网站、民宿和门票；
  2. 35岁以下人群更关注旅游；
  3. 年轻人相对关注民宿，年长者相对关注酒店，社会中坚群体更关注OTA和机票；
  4. 华中相对更关注酒店，华东相对更关注民宿、网站，华北相对更关注机票。

3. 十点读书旅游业务尝试

（1）投资

投资旅游公众号在「十点读书」公众号中，有一篇文章《从0到1000万粉丝，“十点读书”的坚持与梦想》，文中提到十点读书2017年获得投资后投资了其他新媒体公司，其中包括“旅物LIVE”。

“旅物LIVE”这个公众号主要分享游记攻略、销售各地特产，这是十点读书涉足旅游行业的一个尝试。

四、十点旅行号突围点

1. 用户属性

在旅游号分析中，我们可以发现用户属性差异性不大，年龄在18-34岁，主要是25-29岁，宅男宅女居多，女性占大多数。

因此十点旅行号可以25-29岁女性为主，向上辐射到18-24岁大学生及年轻白领，向下辐射到30-34岁的职场熟女和家庭主妇。

18-24岁女性的经济实力一般，但业余时间充足，对于新鲜事物的接受程度更高，因此关注十点旅行号的概率更高、时间更多、分享传播可能性更大。30-34岁的女性已经具备一定的经济基础，对生活品质有自己的追求，而且这类女性大多开始组建家庭，可以带动家庭消费。

十点读书的用户属性与旅游号用户属性吻合，所以建议十点旅行号前期可以充分利用十点读书现有的用户资源，从十点读书进行导流。

2. 内容属性

目前6个旅行公众号覆盖了游记攻略、人物故事、新潮小众、独家路线、旅行介绍、机票酒店和特色美食。但是每个公众号在内容方向上都分别做了细分，各有侧重。

  • 「Feekr旅行」提供小众路线和独家预订
  • 「走着瞧旅行」借明星和影视节目为热点做圣地巡游
  • 「人间旅行指南」主攻江浙一带周边旅游
  • 「稻草人旅行」推荐国内外、长短途的旅行路线
  • 「极简旅行」介绍全球旅行景点

根据竞品分析和突围点分析，整理出十点旅行号的better-worse模型：

3. 魅力属性

而新潮小众、独家预订、旅游社会热点新闻、视频则是各个公众号的特色；十点旅游号可以延续十点读书温暖、文艺的风格，以图文+音频+短视频的形式呈现。

4. 无差异属性

1. 用户来源

（1）公众号创办初期不需要每日更新，固定几天推送文章，养成用户阅读习惯即可。

（2）每日发2-3条以确定内容是否迎合了用户的兴趣和口味，根据阅读情况适当对内容方向、文案图片进行调整，待风格稳定后再根据用户体量增加文章数量。
