网页设计要用到哪些软件?

FLASH动态图象，视频处理，这个不用多说。
PHOTOSHOP图片处理
Dreamwaver传统的专业网页网站制作软件。
还有一个相片处理的软件Macrmedia Fireworks ，处理成品相片特别是网页图片链接这一块比PHOTOSHOP要好点。
一些网页的制作链接，主页和分页的制作。在Dreamwaver都可以实现。不会的话上面会有新手教程。
制作好了之后还有一个网站上传的软件，这个太多，你可以自己去搜一下，各种软件都是不错的。

首先你要申请到一个网站域名，然后再到服务器上申请一个空间，把这个空间和你的域名绑定。然后按照提示。把你的网站源文件上传到指定的空间系统文件夹中。这样。你就可以访问你的网站了。
最好，你先学一下制作网站后台，如果你学会了这个，那么这些软件基本上用不着。只是图片处理一下，需要改动的网站格局在你的后台就可以改动。也更加便于你的管理。

制作网页可以用什么软件？

制作网页可以用如下软件：

1、Coolsite360。特点有可复制模板，更可任性DIY，无需编程，在线创建，自动适配桌面及移动端。

2、Adobe Dreamweaver。简称“DW”，中文名称 "梦想编织者"，最初为美国MACROMEDIA公司开发，2005年被Adobe公司收购。

3、Flash。是由macromedia公司推出的交互式矢量图和 Web 动画的标准，由Adobe公司收购。做Flash动画的人被称之为闪客。

4、Fireworks。是由Macromedia推出的一款网页作图软件，软件可以加速 Web 设计与开发， 是一款创建与优化 Web 图像和快速构建网站与 Web 界面原型的理想工具。

5、Frontpage。Frontpage，是微软公司出品的一款网页制作入门级软件。FrontPage使用方便简单，会用Word 就能做网页，因此相对Dreamweaver等软件更容易上手。

投稿plos one居然告诉我没找到editor？怎么办

没找到editor，也许编辑手头工作太多；对你的工作不熟悉；对你的工作不敢兴趣（可能是质量因素）都是有可能的，

编辑给你回邮时，再建议3个审过相关领域文章的editor不过这次你要谨慎一点，多了解了解该领域有哪些人是PLOS ONE主编。一个办法，你不是说PLOS ONE上也有该类型的文章吗，去它们文章的首页看负责这篇文章的academic editor是谁

如果能等，建议多等一下。否则，投同级水平的专业杂志，你还是要找编辑和审稿人，还是要时间。
