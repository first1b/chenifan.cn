翻车？罗永浩直播间被指卖山寨鞋，日本厂商未授权|电商|微博

小澈 互联网资讯 发布时间：2020-12-25 0

（原标题：又翻车？罗永浩被指直播间卖山寨鞋）

更新：目前，京东RVG X STM旗舰店已下架所有商品，该店铺运营方即交个朋友直播间所售山寨潮鞋的供货商福建昊憬，而消费者反馈尚未收到交个朋友直播间官方回复。

卖假羊毛衫风波未平，近日又有网友爆料称，在罗永浩直播间购买的潮鞋疑似为山寨货。该网友表示，自己在老罗直播间购买了一双RVG X STM潮鞋，外观与日本Revenge X Storm潮鞋相似，但经过第三方鉴定APP鉴定为假货。

对此，该产品直播供货方表示RVG X STM就是Revenge X Storm品牌的短版缩写，第三方鉴定平台识别鉴定是以美版为准，而品牌生产的是日版产品。

不过，据凤凰网电商研究院报道，日本品牌方与罗永浩销售的潮鞋并无授权关系。

商标授权书来自杭州公司

而非日本品牌方

据了解，该鞋在抖音小店RVG X STM昊憬专卖店售卖，12月16日上架罗永浩直播间。数据显示，直播间券后价格219元，产品预估销量1516双，在直播间的销售额超过33万元。同时这款鞋也曾上架李诞直播间，预估销量256双。

罗永浩直播间销售的RVG X STM鞋

据凤凰网电商研究院报道，该鞋的生产厂家为福建昊憬鞋服有限公司。对于“假冒”日本产品，该公司称第三方鉴定平台识别鉴定是以美版为准，而品牌生产的是日版产品。但其出示的“日本Revenge X Storm商标授权书”等材料显示，这款潮鞋获得的是杭州多普商贸注册的“RVG X STM”商标授权，鞋子外观设计专利实际来自于多普商贸在国内申请的“Revenge X Storm外观设计专利”。日本Revenge X Storm官方与多普商贸并无授权关系。

在中国商标网上，与Revenge X Storm相同及近似的商标数量多达256件，其中大多数申请失败或尚未完成申请。多普商贸也曾想注册名为“Revenge X Storm”的完整商标，但注册失败，商标被判定无效。

日本售价约合人民币1800多元

罗永浩直播间卖219元

在Revenge X Storm品牌日本官方网站，商家经常提醒消费者注意识别正品。这款潮鞋在日本售卖价约合人民币1800多元，而在罗永浩直播间，售价仅为219元。

12月24日，职业打假人王海在微博询问消费者有没有上当受骗。之前王海曾揭露罗永浩售卖假羊毛衫，后被罗永浩“交个朋友”直播间证实销售羊毛衫确为假货，目前正在进行退一赔三的补偿。

而在经历假羊毛衫事件后，12月21日“交个朋友”微博称，交个朋友直播间将进行全面反思，决定进行全方位整改升级。措施包括成立“质控实验室”、 与国际知名的第三方专业检测机构SGS开展更为紧密的合作，同时在直播间安排“神秘顾客”+公司内部成立“客户卓越体验委员会”、 积极参与权威机构主导制定的直播电商商品质量检测和控制相关行业标准和规范、面向行业招募大牛和质控专家等。

对于罗永浩声称的“全面反思、整改”，在王海看来只是为了应付舆论。王海向红星新闻表示，后面还会继续追究下去，直到罗永浩进行赔偿。

目前罗永浩还没有正面回应假鞋问题，红星新闻联系罗永浩微博进行询问，截至发稿未收到回复。不过今天中午罗永浩在微博称，很多媒体和网民的最大魅力在于，自己天天在100%有假货的电商大平台买东西，但直播行业出了些类似问题，他们就“惊诧莫名”，即使某些直播从业者在出问题时，比任何电商大平台出现同样问题时解决得都更快更好更有诚意，他们也继续惊诧莫名。
