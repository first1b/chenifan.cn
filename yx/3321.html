用视频号“日赚百万”，这些方法绝了

[db:文章来源] 营销 发布时间：2021-01-26 0

今年一年以来，视频号前前后后进行了十几次改版。

从最初的可以转发到朋友圈，到最近刚推出的「推流」功能，“布局视频号”已经成了自媒体行业的共识。

不过，大家平时讨论比较多的内容更多的是怎样引流、怎样把号做大，很少有人去认真想，不同类型的视频号，怎样变现是最高效，而且是可持续变现的呢？

其实，不仅是头部账号，中小视频号的变现，也早就可以开始做起来了。

今天，我们就来探讨一下，不同类型的视频号，都有哪些适合的变现手段呢？

01

内容 IP 变现

什么叫内容 IP 呢？就是内容比「做出内容的人」更出名。

你可能知道他做出过哪些爆款内容，但你未必叫得出作者的名字。

前几天因为「直播流星雨」而一夜爆红的 @李政霖，就是这种“内容 IP ”的典型代表。

李政霖的视频号全是风景，自己很少出镜

那么，类似这种内容 IP 最适合的变现路径是哪些呢？

1）广告变现

首先，对于内容关注高的账号来说，最直接、最恰当的变现方式当然是视频号内接广告。

一般来说，在视频号接广告主要有这几种形式：

① 视频号下挂链接

大家都知道，视频号下方是支持链接公众号文章的，视频播放多少次，这个链接就会有多少次曝光。

因此，这个位置其实就是可以作为一个广告坑位出售的，你交坑位费、我挂你的链接，帮你引流。

比如说，下面这位情感向的动画博主，突然接了一个八竿子打不着的“中小学单词课”，就很可能是收了坑位费的操作：

 

当然，这种广告也不一定是按照坑位费结算的，毕竟视频号现在刚起步，大家普遍数据不高。

为了保证效果，现在也有不少商家喜欢通过 CPS 广告的方式进行合作。

② CPS 广告

CPS 是啥呢？其实就是「按销售付费」，再通俗点说就是“赚佣金”。

比如，我在你这里投放一个理财课程的链接，同时给到一个渠道码监测，通过你的渠道一共转化了多少金额，按照比例来结算广告费用。

当然，不管是坑位费还是 CPS ，这种接广告的机会基本是甲方爸爸主动找你才有的。

如果没有的话，自己也可以找一些方法进行变现。

比如，刚刚那个动画号，经常在视频底下挂了一个“查查全国有多少人和你重名”的公众号链接。

点进链接后，直接引导跳转到一个付费的第三方查询平台，只要你在这个入口进行了付费，那号主就可以赚到佣金。

这种平台还有很多，比如在线占卜的、在线起名的，都是可以申请成为分销员的，赚佣金的：

 

不过，这种链接很多都涉及到灰黑产，大家在操作的时候一定要见机行事，避免被割韭菜……

③ 定制视频内容

定制视频号内容大家都比较熟悉，在微博、B站等平台也早有先例。

不过，由于视频号现在刚刚起步，因此这种形式的合作比较少，一般只有头部视频号才有这种机会。

比如，头部情感公众号@灵魂有香气的女子的创始人李筱懿，就曾在视频号接过怡丽思尔等品牌的定制广告。

 

至于价格嘛，可以给大家一个参考，据称@夜听创始人的视频号广告对外报价是 20w 一条……

2）引导至公众号变现

很多输出内容 IP 视频号，本身就是脱胎于自己公众号的。

这时候，我们就还可以选择将通过视频下方的链接，配合文案引导粉丝关注公众号，再在公众号进行转化。

 

用这种方式，不仅能帮公众号涨粉，同时也可以让视频号产生更大的收益。

① 流量主功能

首先，第一份收入来自公众号的「流量主」功能。

开通这个功能后，只要有用户点击广告位出现的广告，就可以获得收益：

 

这个功能属于被动收入，可以直接把文章阅读量变现。

不要小瞧视频号的引流能力。像公众号@NOTHINGART 就通过这个功能，直接把单篇文章的阅读量翻了近 4 倍：

更极端一点的，公众号@蝌蚪课，平时的头条阅读量不过万，爆款视频下链接的文章阅读数直接上了 10w+ ：

 

一般来说，10w+ 的文章，保守估计光流量主收入就能有 2k 左右。

而且，@蝌蚪课的爆款视频非常多，虽然不能做到篇篇 10w+，但是 5-6w 阅读的文章比比皆是。

因此，光是流量主的收益，就让很多团队望尘莫及了……

② 插入商品卡片

对于一些没有能力同步进行公众号更新，只做视频号的内容中小号号主来说，我们的变现方式可以简单粗暴一点。

首先，通过在京东 or 拼多多申请成为分销员，获得商品的分销链接。

然后，直接把引导文案设置成“点击购买同款商品”，省略掉公众号文章的文字部分，再在文章里插入商品链接，这样就可以利用流量变现啦：

 

02

个人 IP 变现

如果说内容 IP 是以内容为主，用户主要关注的是内容本身，而非“生产内容的人”；

那么，个人 IP 则是给到生产内容的人更多曝光，让用户对这个人产生信任感，号主的号召力很强。

由于个人 IP 的内容一般更加垂直，因此，相比内容 IP ，个人 IP 吸引来的用户往往需求更强烈、相对来说粘性更高。

对于个人 IP 而言，内容 IP 们常见的接广告、挂链接模式也都可以走：

比如个人 IP @小小包麻麻，就经常将卖货文案发布在公众号，再通过视频号给链接引流来变现。

 

但是，这样就没有发挥出视频号最大的优势 ——

本身在微信生态内，和公众号、社群、小程序等功能打通十分方便。

所以，对于个人 IP 的视频号在做变现时，最适合的变现方法反而是“不急着变现”，先将视频号的流量导入私域，再在私域进行爆发式转化。

最好的情况，是可以形成「闭环运营」，也就是把视频号和私域（公众号、小程序、社群）联动起来：

 

1）本身无私域的：先沉淀成私域，后变现

对于本身还没有沉淀出私域流量的个人 IP 来说，可以将用户先沉淀到视频号再变现。

可以选择在视频号主页、评论区、底部公众号文章里放上微信号，再用文案告诉大家“加微信有福利”，以此引导大家添加：

 

同时，还可以在主页引导用户私信你，并在私信回复微信二维码：

 

引导用户私信的好处是，二维码长按就可以添加，不用用户复制去搜，加好友入口更浅，路径也更通畅；而且，还有机会再次介绍自己的资料包，进一步打动用户添加好友。

不过，这种方法也有缺陷 —— 目前视频号还没有自动回复私信的工具，如果要使用这种方式，就需要挨个手动回复用户了，比较麻烦。

通过从视频号引流到私域，曾在抖音、快手运营过同名 IP 的视频号 @时光雕刻大师傅，在仅有 10 人小团队的情况下，用了几个月时间，积累了 8.8 万粉丝，并通过老照片修复业务变现了 30 万元。

 

其主创人员周华曾在@新榜 的采访中提到过，和抖音快手的年轻用户相比，视频号的粉丝多是 35+、有一定经济基础的中年人，购买意愿更高，私域积累更快，变现也相对更容易。

不过，值得注意的是，不建议大家直接把联系方式直接内置在视频里。

虽然目前视频号是可以直接插入微信 / 企微二维码的，暂时不会出现被屏蔽、封号这种“恶性后果”；但是，据业内人士爆料，在视频内直接插入二维码有被限流的风险。

所以，还是建议大家在公众号链接 or 评论区进行引导，不仅沉淀路径更加通畅，风险也小一些。

2）本身有私域的：激活私域，提高转化

本身有私域的个人 IP，在视频号的变现应该是最有利的：

① 直播带货

如果本身就有一定私域流量的积累，这时候，就可以用视频号这个新场景，来实现更多的变现可能。

基于用户对主播本人的高信任度，相比接广告、带链接卖货，有私域的个人 IP 更值得尝试的是 —— 视频号直播带货。

为什么这么说呢？

因为，可以在私域流量先做启动，到时候来看直播的用户，基本都是对主播信任度高的高粘性用户。

同时，相比图文链接，直播带货的煽动性更强，用户下单的可能性也更大，可以让每个用户在生命周期内给你带来更大的价值。

两者结合起来，效果可能相当惊人。

比如，视频号的头部个人 IP @小小包麻麻，通过提前在社群预热，结合用海报做裂变的玩法，在直播间带货超过 180 万。

刚刚我们提到过的 @李筱懿，第一场直播带货的数据也很出彩：

观众 6.5 万人，成交单数近 2 万，总成交金额 190 万。

和@小小包麻麻一样，在直播前，@李筱懿也在公众号、社群、视频号做了相当充分的预热：

① 公众号：发布直播预告，并将预告图文放在公众号菜单栏里持续引流； ② 视频号：发布直播预告视频； ③ 社群：另外，在 77 个城市社群里进行宣传，为直播引流。

图源新榜

预热后，将想看直播的用户汇集到 28 个社群里，通过群名和群消息提醒用户观看直播；同时，还能在群里引导关注视频号，给视频号导流，进一步涨粉。

03

品牌 IP 变现

作为品牌，就意味着本身已经有一定的核心业务，比如电商、知识付费、食品、美妆等。

现在，入局视频号最多的就属 K12 、知识付费等教育类账号了。

不过，品牌 IP 的“变现”注重的并非当下视频号带来的直接收益，而是将它作为「新的流量来源」，利用它将本来的业务做得更大。

因此，品牌 IP 视频号变现的核心指标，往往是增加曝光，以提高整体转化。根据这个指标出发的变现手段，主要有这两种：

第一种就是用内容精准的视频号内容，给低价体验课（一般客单价低于 10 元）引流。

 

将购买过课程的用户引流到私域后，再做精细化运营，通过课程质量和服务打动用户，促使用户进行二次、三次的付费。

第二种，就是通过更泛的视频号内容吸引大量流量，再通过直播、社群等各种手段反复触达达成转化，运营手段比较粗放。

比如，下面这个号主，本身是某英语机构的老师，这个机构的公众号今年 9 月才注册，阅读量很一般，平均阅读量在 2k 左右：

 

但是，它孵化的视频号经常会发一些追热点、情感类的内容，因此个人 IP 平均点赞量在 1w 上下， 5.6w 点赞不罕见，甚至还有好几条 10w+ 出现：

 

可以说，视频号成了他们最大的流量来源。

同时，他还会在视频号进行每天 2 次的课程直播，通过「关注视频号，听免费直播课」的引导，将用户沉淀到自己的私域流量池；同时，在直播课的最后，也会通过口播进行卖课。

04

结语

说了这么多，是不是觉得视频号的变现一点也不难？

不管是大号还是小号、粘度高还是粘度低的流量，都有办法进行变现。

相信未来，视频号还会开放更多有利引流变现的功能，比如上次张小龙在@潘乱 的直播间说的“下次来连麦”，以及已经在内测中的付费推广视频号等功能。

视频号的未来，还有很多种可能。
